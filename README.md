#### 项目组成
项目由三个模块组成：物资监控、业务监控、告警监控
图表：echart
屏幕自适应：lib-flexible（需要手动配置）
api：axios
轮播：vue-awesome-swiper
数字翻页：vue-count-to
无缝滚动：vue-seamless-scroll
事件通信：$Event
UI：element-ui
地图：腾讯地图、热力图
